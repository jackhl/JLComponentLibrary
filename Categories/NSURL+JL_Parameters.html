Class Methods

JL_URLWithProtocol:domain:endpoint:parameters:

Creates an NSURL by formatting and concatenating commonly dynamically joined URL components.

+ (instancetype)JL_URLWithProtocol:(JL_URLProtocol)protocol domain:(NSString *)domain endpoint:(NSString *)endpoint parameters:(NSDictionary *)parameters

Parameters

protocol

The protocol to use.

domain

The domain to place the request on. Do not end the domain with a forward slash.

endpoint

The endpoint to add to the end of the domain. Do not prefix the endpoint with a forward slash. Pass nil if the URL is simply the domain.

parameters

A dictionary of key-value pairs to append to the URL by adding a question mark (?) to the end of the URL and then separating each key from each value with an equal sign (=) and each pair with an ampersand (&). Pass nil if the URL does not take any parameters.

Return Value

A URL by joining the components passed in as method parameters.

Discussion

Components are joined like so: <protocol>domain/<endpoint>?key1=value1&key2=value2
